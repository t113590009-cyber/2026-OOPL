# Game Introduction

本專題要復刻的遊戲是 **《貓咪大戰爭》(The Battle Cats)**。  
遊戲的基本流程如下：

1. 玩家隨時間累積金錢(或透過擊殺敵人取得金錢)
2. 使用金錢召喚貓咪單位
3. 貓咪與敵人自動向前移動
4. 當單位進入攻擊範圍時開始戰鬥
5. 摧毀敵方主塔即可獲勝

主要遊戲元素包括：

- 貓咪單位 (Player Units)
- 敵人單位 (Enemy Units)
- 主塔 (Base)
- 金錢系統
- 產兵冷卻時間
- 遊戲外升級與進階系統

[遊戲連結](https://www.youtube.com/watch?v=qHGv7VjKGJU)
