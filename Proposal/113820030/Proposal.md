# Game Introduction

這是一款經典的橫向卷軸街機遊戲。玩家透過點擊滑鼠或按下按鍵，控制不斷向前飛行的小鳥，藉由拍打翅膀來維持飛行高度，並閃避沿途高低不一的水管障礙物。遊戲考驗玩家的反應速度與節奏感，只要稍微失誤碰到水管或掉落地面，遊戲就會立即結束。

實作功能：

1. **基本物理與控制**：實作小鳥受到重力影響自動下墜，以及點擊時的向上跳躍。
2. **障礙物自動生成**：遊戲畫面會不斷向左捲動，並隨機生成高度不一的水管缺口供玩家穿越。
3. **碰撞偵測 (Collision Detection)**：當小鳥的邊界觸碰到水管邊緣、地面或畫面頂端時，判定遊戲結束。
4. **計分與最高分系統**：成功安全穿越一對水管即得一分，並可在本地端記錄玩家的歷史最高分數。
5. **遊戲狀態場景**：包含「首頁開始畫面 (Start Screen)」、「遊戲進行中 (Playing)」與「遊戲結束/重新開始畫面 (Game Over)」。
6. **多媒體效果**：加入背景音樂、跳躍/得分/碰撞音效，以及小鳥飛行的漸變動畫與視差捲動背景。
