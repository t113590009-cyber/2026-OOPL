# Game Introduction

***
這是一款 2D 策略塔防遊戲。玩家必須在路徑旁放置四種不同的防禦塔，玩家不只能蓋塔，還能操作英雄單位在地圖上自由移動支援，並手動施放法術

此外玩家必須針對不同敵人的護甲屬性（物理護甲或魔法抗性）建造不同的防禦塔，也可以利用兵營阻擋敵人的行進路線。



[遊戲連結](https://www.youtube.com/watch?v=Dy13H--bDgo&list=PLEszIV31eZRY_oxfiMBdaEh8ZgYO9vrca)
