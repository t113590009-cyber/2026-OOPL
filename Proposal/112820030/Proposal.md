# Game Introduction

是一款結合輕度 Roguelite 元素與休閒操作的 2D 俯視視角時間生存遊戲。玩家控制角色在成群怪物圍攻中，透過自動攻擊、升級選擇，盡可能存活。

[遊戲連結](https://www.youtube.com/watch?v=bzYEU3rBD-Y)
