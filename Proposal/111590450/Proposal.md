# Game Introduction
《Bloons Tower Defense》是一款經典的單機塔防遊戲。玩家需要在地圖上放置防禦塔，阻止一波波氣球敵人沿著固定路線前進。透過消滅氣球獲得金錢，玩家可以建造更多防禦塔或升級現有塔，逐步建立更強大的防線，抵禦越來越強的敵人。

* **核心玩法**：建造防禦塔、阻止敵人通過路線、管理金錢與升級策略。
* **遊戲機制**
  * 敵人沿固定路線前進
  * 防禦塔自動攻擊範圍內敵人
  * 擊敗敵人可獲得金錢
  * 玩家可以升級塔或建造新塔
  * 隨著波次增加敵人難度提高

* **視覺風格**
  * 2D 卡通風格塔防遊戲
  * 使用簡單圖形區分不同敵人與塔
