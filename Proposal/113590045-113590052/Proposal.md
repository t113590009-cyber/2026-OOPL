# Game Introduction
---
遊戲內容含主篇章（世界篇、未來篇、宇宙篇、魔界篇）、傳奇系列（傳奇故事、真傳奇故事、傳奇故事0）、特別關卡、活動關卡、道場等地圖。玩家需先在前期收集主篇章的寶物以強化貓咪的能力，接著逐步於以週或月為週期舉辦的各種特別關卡（如漩渦系列、狂亂系列、極難系列、開眼系列）之中取得更多免費的貓咪或其更高形態進化的權利的同時，在週期舉辦的特別關卡（經驗值系列、貓薄荷系列、超獸討伐/獸石系列、發掘系列）獲取養成角色所需的升級或進化素材，在通過養成角色獲得更高戰力後於傳奇系列及貓咪道檢定中取得更多進展。玩家可同時通過各種期間限定（節日（如新年）、慶典（如日版13週年）、合作（如Eva(新世紀福音戰士)等）活動獲取更多額外的素材與限定角色。
